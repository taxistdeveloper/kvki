# Контент страниц

## Два типа файлов

### 1. `.html` — контент из админки
- Создаётся при сохранении в админ-панели
- Чистый HTML, вставляется в шаблон сайта

### 2. `.php` — шаблон, слитый с сайтом
- Создайте вручную для полного контроля
- Имеет доступ к `$menu`, `$router`, `$pageData`, `$pageTitle`, `BASE_URL`
- Использует `page-layout.php` — header, footer, стили сайта

## Пример (content/pages/тест.php)

```php
<?php
$content = '
<h1>Заголовок</h1>
<p>Контент страницы в HTML.</p>
';
require ROOT_PATH . '/templates/page-layout.php';
```

## Приоритет
- `.php` — если есть, используется (полный шаблон)
- `.html` — иначе контент из файла
- БД — иначе из базы данных
